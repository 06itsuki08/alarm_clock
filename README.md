# alarm_clock

2020 年度の卒業研究のアプリです。
元々は 2018 年度の研究の改良版

作成：チーム卒研猫の会

## Used Plugin

- flutter_launcher_icons
- flutter_local_notifications
- shared_preferences

使用用途を上から、

- アプリのアイコンを一括で変更する
- アプリの通知
- 端末に設定データを読み書きさせる

## Tools

- VSCode
- AndroidStudio
- Github
- GoogleDrive とその仲間たち
